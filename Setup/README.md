# My Custom Setup

### The Color-Scheme came from [Materialize](https://github.com/saadq/Materialize/blob/master/schemes/Material%20Vim%20Blackboard.tmTheme) by [saadq](https://github.com/saadq). Check it out.

#### Step 1-) Install Sublime Text as the text editor of choice
#### Step 2-) Install the files needed to run Python and C++
#### Step 3-) Make sure to have everything up and running.
#### Step 4-) Add the [Material Vim Blackboard.tmTheme](https://github.com/jiakangchen005/Competitive-Programming/blob/main/Setup/Material%20Vim%20Blackboard.tmTheme) to the /Sublime Text/Packages/User/ directory somewhere depending on the OS.
#### Step 5-) Sublime Text Tab > Preferences > Color Scheme > Material Vim Blackboard.tmTheme
#### Step 6-) Add whatever you want. Preferably useful features and key-bindings.
#### Step 7-) Done. Enjoy.

### Also, the Default.sublime-keymap file requires specific files to run properly, mainly from my [templates](https://github.com/jiakangchen005/Competitive-Programming/tree/main/templates) folder.
